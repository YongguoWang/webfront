### less
 - 概念： less是一个预编译处理语言，是css的拓展，它让我们的css更具有维护性，主题性，拓展性。

 - 安装环境
 步骤： 1. 下载nodejs软件
        2. win + R  输入  cmd  , 打开命令面板
        3. 输入 node -v   显示版本号，表示nodejs软件安装成功
        4. 输入 npm  -v 显示版本号， 表示可以使用npm 包管理工具
        5. 使用 npm(包管理工具)  下载 less
        6. 输入 lessc -v  显示版本号 ，安装成功了。

- 命令面板

- ctrl + c  表示终止命令

- cls     清屏

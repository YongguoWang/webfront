### 今天内容

- 盒子模型
1. 外边距（margin
    + 设置盒子的位置。
2.边框（border
    + 盒子有四个边框，边框会撑开盒子大小。（边框:线宽  线样式 线颜色）
3.内边距（padding
    + 设置盒子内容与盒子边框距离。内边距会撑开盒子大小。
4.内容（width/height

- 盒子大小的计算方式

- 盒子大小：边框 + 内边距 + 内容

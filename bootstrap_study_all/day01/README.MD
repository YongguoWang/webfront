### Bootstrap (响应式开发)
1.0 流式布局（百分比布局）
- 同一个页面在不同的设备屏幕下，都显示同一个版本。

2.0 响应式布局
- 同一个页面在不同的设备屏幕下, 显示的是不同的版本。（超大屏幕 ，中等屏幕 ， 小屏幕 ，超小屏幕）

###### 响应式的原理
- 根据媒介查询属性，监听设备屏幕宽度，从而让页面作出相应的变化。
@media () { }
@media screen and () { }
@media () and () { }
